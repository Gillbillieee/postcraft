# PostCraft — AI Social Media Post Generator

A free, client-side tool for generating engaging LinkedIn & Twitter posts. No signup required.

## Features

- 🚀 Generate 3 unique post variations instantly
- 🎭 5 tone options (Professional, Casual, Contrarian, Storytelling, Humorous)
- 📱 Platform-optimized formatting (LinkedIn + Twitter/X)
- 💾 Auto-saves drafts to your browser
- 📋 One-click copy & download
- 🎨 Beautiful dark-mode UI

## Tech Stack

- HTML5 / CSS3 / Vanilla JavaScript
- Zero dependencies, zero build step
- Deployable anywhere (Vercel, Netlify, GitHub Pages)

## Quick Deploy

### Option 1: Vercel (Recommended)
```bash
# Install Vercel CLI
npm i -g vercel

# Deploy (requires logged-in Vercel account)
vercel --prod
```

### Option 2: Netlify
```bash
# Install Netlify CLI
npm i -g netlify-cli

# Deploy from the project directory
netlify deploy --prod --dir .
```

### Option 3: GitHub Pages
1. Push to a GitHub repo named `postcraft`
2. Go to Settings → Pages → Source: main branch
3. Your site will be at `https://yourusername.github.io/postcraft/`

### Option 4: Drag & Drop (Netlify)
1. Go to app.netlify.com/drop
2. Drag the entire project folder onto the page
3. Done — live in seconds
